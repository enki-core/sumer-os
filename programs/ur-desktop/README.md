# 🌌 Sumer OS Portable (نظام سومر المحمول)

نظام تشغيل محاكى ومحمول مبني باستخدام **Rust** ومكتبة **Slint 1.5.0** لتوفير واجهة مستخدم حديثة، سريعة، وتدعم اللغتين العربية والإنجليزية بشكل متكامل مع دعم كامل لاتجاه اليمين-إلى-اليسار (RTL).

---

## 🏛️ البنية المعمارية (Architecture)

يعتمد النظام على نمط **مركزية البيانات (Centralized State)** حيث يعمل الـ Rust كخلفية (Backend) تدير الحالة المنطقية، بينما تعمل مكونات Slint كواجهة عرض تفاعلية (Frontend).

### 📂 هيكل الملفات والمكونات

```
sumer-ui/
├── src/
│   └── main.rs                 # المحرك الأساسي بالـ Rust وإدارة المهام غير المتزامنة (Async Tasks)
├── ui/
│   └── main-desktop/
│       ├── models.slint        # تعريف هياكل البيانات (AppIcon, Point)
│       ├── theme.slint         # المرجع المركزي للألوان، الأنماط، الظلال، والحركات (Design Tokens)
│       ├── desktop.slint       # المكون الرئيسي لسطح المكتب وإدارة النوافذ والـ Dock
│       ├── desktop-background  # الخلفية التفاعلية والتوهج الديناميكي (Radial Glow)
│       ├── desktop-header      # الترويسة العلوية لسطح المكتب
│       ├── desktop-icons-layer # طبقة أيقونات سطح المكتب وإدارة السحب والإفلات والـ Evasion
│       ├── desktop-icon        # مكون الأيقونة الفردية مع دعم ألوان الـ Models
│       ├── desktop-windows-layer # طبقة النوافذ النشطة وغير النشطة وشريط النوافذ المصغرة
│       ├── window-frame        # إطار النوافذ (شريط العنوان، الإغلاق، التكبير، والتصغير)
│       ├── taskbar             # شريط المهام العائم (Dock)
│       ├── start-menu          # قائمة ابدأ التفاعلية
│       ├── control-center      # مركز التحكم (واي فاي، بلوتوث، إضاءة، صوت، مشغل الميديا)
│       ├── system-tray         # شريط النظام (الساعة، البطارية، الشبكة)
│       ├── file-manager        # تطبيق مدير الملفات التفاعلي (الأرشيف السومري)
│       ├── console-window      # تطبيق موجه الأوامر المحاكى (شيل دجلة)
│       └── drag-drop-manager   # مكتبة الرياضيات وحسابات السحب والـ Evasion (SumerDragDropMath)
```

## 🚀 الميزات الرئيسية

1. **دعم كامل لـ Slint 1.5.0:** استخدام أحدث ميزات Slint مثل `Flickable`، `TouchArea`، والتصدير المباشر للمكونات.
2. **نظام تصميم مركزي (Theme Tokens):** التخلص من الألوان المشفّرة والاعتماد على `SumerTheme` لتوحيد الهوية البصرية.
3. **توجيه عربي أصيل (RTL):** محاذاة كافة المكونات والقوائم والنصوص لتتناسب مع اللغة العربية بشكل طبيعي.
4. **تطبيقات تفاعلية حقيقية:**
   * **مدير الملفات:** تصفح مجلدات حقيقية ومعاينة نصوص مسمارية وسومرية.
   * **شيل دجلة:** محرك REPL تفاعلي يدعم أوامر `sysinfo`, `neofetch`, `help`, `clear`.
5. **فيزياء تفاعلية متقدمة:**
   * نظام قفز الأيقونات (Evasion Logic) محسّن لعدم التداخل.
   * إمكانية إعادة الأيقونات من الـ Dock إلى سطح المكتب عند سحبها للأعلى.
   * حماية النوافذ من الخروج خارج حدود الشاشة (Window Bounding).
   * تمييز النافذة النشطة بإطار سماوي متوهج.
   * شريط خاص بالنوافذ المصغرة (Minimized Windows Pill) لاستعادتها بضغطة زر.

## 🛠️ التشغيل والتطوير

لتشغيل المشروع في بيئة التطوير المحلية:

```bash
cargo run
```

للتحقق من سلامة الكود والبناء:

```bash
cargo check
```
